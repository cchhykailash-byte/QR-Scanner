# 🔗 QR Code Generator

A modern QR Code Generator built using Streamlit.

Generate beautiful QR Codes from

- Websites
- Social Media
- WiFi
- Contact Info
- Text

Features

- Rounded QR
- Dot QR
- Square QR
- Custom Colors
- Custom Background
- Logo in Center
- PNG Download
- JPG Download
- Mobile Friendly
- Instant Preview

## Installation

```bash
git clone https://github.com/USERNAME/qr-code-generator.git

cd qr-code-generator

pip install -r requirements.txt

streamlit run app.py
```

---

## Deploy on Render

Build Command

```bash
pip install -r requirements.txt
```

Start Command

```bash
streamlit run app.py --server.port=$PORT --server.address=0.0.0.0
```
